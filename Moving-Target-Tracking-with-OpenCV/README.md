# Installation and use

Requirements:
1. JDK 11
2. Eclipse Java IDE
3. JUnit 4.12
4. OpenCV 4.0.0
5. Source folder

Installation:
1. Ensure build path references JUnit 4.12 and OpenCV 4.0.0
2. Open source folder with Eclipse, choosing the master directory as the project file
3. Run file main, adjust input as needed
